### 目前状况

**制作更新中**

## 如何部署

1. git clone 这个项目

> git clone https://github.com/Miya-Yukarin/Yukarin-Carnvial-HomePage.git

2. 安装依赖

> cd Yukarin-Carnvial-HomePage && npm install

3. 部署展示

> npm start
